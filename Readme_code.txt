Requirements:
For running the "test" python script, a Linux based platform (Ubuntu 22.04.2 LTS) is required. 
Before executing any code/exe program grant read/write permission to "IAR-Net-Code"
>>> chmod -R 777 IAR-Net-Code
######################################################################
Follow the steps below for running the scripts.

1. Ensure that all the dependencies are installed. 
2. The spatio-temporal video frames for testing are located in the "Input" folder.
3. Open terminal in the current working directory and execute the following code segment.
>>>python test.py 
4. Wait for the results to be generated in the "Results" folder.
